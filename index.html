QUICK START

Windows Clash 怎么用？

不需要修改复杂参数。准备好 Windows 电脑和边界云订阅，按下面顺序完成安装与连接。

01

注册并获取订阅

登录边界云用户中心，选择流量套餐后复制专属订阅地址。订阅链接等同于账户凭证，请勿公开分享。

前往边界云注册 ↗
02

下载 Windows 客户端

前往边界云下载中心，获取适用于 Windows 10 / Windows 11 的 Clash 客户端并完成安装。

下载 Windows 客户端 ↗
03

导入 Clash 订阅

打开 Clash，在订阅、配置或 Profiles 页面粘贴订阅地址，点击导入或更新，等待节点列表出现。

04

选择节点并连接

新手建议先选择自动线路，也可以按延迟选择香港、日本或新加坡节点，然后开启系统代理。

VISUAL WALKTHROUGH

照着界面，一步一步操作

下面以 Windows 版 Clash Verge 为例，从下载到连接验证完整演示。真实截图中的账号、订阅链接、二维码和日期均已脱敏；示意界面会单独标注。

步骤 01 · DOWNLOAD

下载 Windows 客户端

  1. 打开边界云官网的“下载”页面。
  2. 找到 Windows / Clash Verge 客户端。
  3. 普通 Intel、AMD 电脑选择 x64 安装版。
  4. 下载完成后，在浏览器下载列表中打开安装文件。
下载提醒只使用边界云下载中心或 Clash Verge Rev 官方发布页，不要从网盘、论坛或陌生群组下载安装包。
步骤 02 · INSTALL

完成安装并首次启动

  1. 双击下载完成的安装文件。
  2. Windows 弹出确认窗口时核对应用名称与来源。
  3. 按安装向导完成安装，不需要修改高级选项。
  4. 安装完成后启动 Clash Verge。

如果提示缺少 WebView2，请安装微软 WebView2 Runtime，或使用下载中心提供的内置 WebView2 版本。

步骤 03 · ACCOUNT

登录边界云用户中心

  1. 打开边界云官网并点击“登录”。
  2. 进入用户中心，确认套餐状态为“生效中”。
  3. 检查剩余流量和到期时间。
  4. 点击当前订阅的“查看详情”。
隐私提醒公开截图前必须遮住账号、昵称、到期时间、订单信息和其他可识别内容。
步骤 04 · SUBSCRIPTION

复制你的专属订阅地址

  1. 登录边界云用户中心。
  2. 进入“订阅”或“我的服务”。
  3. 找到 Clash 订阅并点击复制。
隐私提醒订阅链接相当于连接凭证，截图前必须遮住完整地址。
步骤 05 · PROFILES

在 Profiles 页面导入订阅

  1. 打开 Clash 客户端，进入 Profiles / 配置页面。
  2. 在订阅地址输入框粘贴链接。
  3. 点击 Import、Download 或“导入”。
  4. 等待配置卡片出现，并设为当前配置。

如果提示下载失败，先检查订阅是否有效，再确认 Windows 时间与网络正常。

步骤 06 · UPDATE

设为当前配置并更新订阅

  1. 在订阅卡片上点击，使它成为当前配置。
  2. 点击右上角刷新按钮更新订阅。
  3. 确认没有出现下载失败或格式错误。
  4. 以后节点发生变化时，也在这里手动更新。

订阅链接失效、套餐到期或本地网络异常，都可能导致更新失败。

步骤 07 · PROXIES

测速并选择合适的节点

  1. 进入 Proxies / 代理页面。
  2. 点击测速图标，等待延迟结果。
  3. 新手可选“自动选择”，也可选择低延迟节点。
  4. 日常使用优先选择香港、日本或新加坡线路。
延迟较低可以使用建议切换
步骤 08 · SYSTEM PROXY

开启系统代理，完成连接

  1. 返回 General / 首页或设置页面。
  2. 将 Mode / 模式设为 Rule / 规则。
  3. 打开 System Proxy / 系统代理。
  4. 访问网页测试；不用时关闭系统代理即可。
✓
完成标志系统代理已开启，节点已选中，网页可以正常加载。
步骤 09 · VERIFY

验证连接并学会正确关闭

  1. 打开浏览器访问常用网页，确认能够正常加载。
  2. 观察 Clash 左下角是否出现上传和下载流量。
  3. 如果无法访问，依次更新订阅、切换节点并重开系统代理。
  4. 不使用时关闭“系统代理”；不要直接强制结束程序。
✓
配置完成订阅可更新、节点已选择、系统代理开启且网页可以加载。
PROXY MODE

规则模式还是全局模式？

对大多数用户，推荐日常使用规则模式。它会自动判断哪些连接需要代理，国内网站保持直连，兼顾速度和流量消耗。

分

规则模式

按规则自动分流，适合网页、AI 工具和日常办公。

推荐
全

全局模式

大部分流量经过代理，适合临时测试和问题排查。

直

直连模式

关闭代理转发，恢复本地网络的直接连接。

TROUBLESHOOTING

Clash 常见故障排查

“已连接但无法上网”“订阅更新失败”“节点延迟高”通常可以通过下面的顺序快速定位。

01

检查系统代理

确认系统代理开关已开启，避免与其他代理软件同时运行。

02

更新订阅配置

检查套餐是否到期、流量是否用完以及订阅链接是否失效。

03

更换可用节点

测试香港、日本、新加坡节点，优先选择低延迟、低负载线路。

04

重启客户端

完全退出 Clash 后重新打开，必要时重启电脑并检查网络设置。

FAQ

Clash Windows 常见问题

覆盖安装、订阅、节点、系统代理和连接失败等常见搜索问题。

01Clash 显示已连接，为什么浏览器仍然无法上网？＋

先确认系统代理已开启，再检查订阅是否过期、节点是否可用。可以切换另一个节点后重试。

02Windows 10 和 Windows 11 都可以使用 Clash 吗？＋

可以。本教程适用于 Windows 10 与 Windows 11，请优先使用下载中心提供的当前版本。

03Clash 订阅链接应该粘贴在哪里？＋

一般在客户端的订阅、配置或 Profiles 页面，粘贴后点击导入或更新。

04香港、日本、新加坡节点应该怎么选？＋

优先选择延迟低、负载低的节点，实际速度还会受到本地网络和高峰时段影响。

05Clash 全局代理和规则模式有什么区别？＋

规则模式会自动分流，更适合日常使用；全局模式会让大部分连接都经过代理。

06如何更新 Clash 订阅和节点列表？＋

进入订阅或配置页面，找到当前订阅并点击更新。不要将完整订阅地址发送到公开 Issue。

07边界云加速器的官方网站是什么？＋

目前仅有 www.bianjiecloud.com 和 www.lvpn.cc 两个官方网站。其他相似域名不要登录、付款或粘贴订阅链接。
